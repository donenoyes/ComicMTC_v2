## 技术点
- 使用ViewModel+LiveData.完成MVVM模式
- 首页多布局+CoordinatorLayout+自定义Behavior，仿有妖气上滑移动特效
- 采用Retrofit2+Kotlin-Coroutines进行网络请求
- RecyclerView预加载
- 等等
